# AIOS Skills

Claude Code skills from the **AI Systems Club**. Free to install — copy, paste, go.

## Install

```text
/plugin marketplace add marcdomancie/aios-skills
/plugin install <skill>@aios-skills
```

Run the first line once to register the marketplace, then install any skill below.

## Skills

| Skill | What it does |
|---|---|
| **codex-image** | Generate images via OpenAI's image model on your ChatGPT/Codex subscription — no API key, no per-image cost. |

### codex-image

```text
/plugin marketplace add marcdomancie/aios-skills
/plugin install codex-image@aios-skills
```

**Requires:** the Codex desktop app (or `codex` CLI) installed, and `codex login` with a ChatGPT plan (Plus / Pro / Team). macOS-oriented; works anywhere `codex` is on PATH.
